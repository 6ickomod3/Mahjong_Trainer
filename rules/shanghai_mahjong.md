# 上海敲麻规则与计分方法

## 1. 核心规则

### 牌具
- 共144张牌：万、筒、条各36张，风牌（东南西北）16张，箭牌（中发白）12张，花牌（春夏秋冬梅兰竹菊）8张。
- **现代变体**：部分玩法将 **中、发、白** 也视为花牌，此时花牌总数变为20张，增加计分变化。

### 基本打法
- 可进行**吃、碰、杠**。
- 庄家起手14张牌，先出牌；闲家13张。
- 第一局随机定庄，之后**谁胡牌谁当庄家**，流局则原庄家连庄。

### 核心特色——“敲”
- **“敲”的定义**：玩家必须公开宣告自己已经**听牌**（如将一张花牌竖起或叠放），这个过程叫“敲”。只有“敲”了之后，才能胡牌（包括点炮和自摸）。
- **“敲”后的限制**：
  - 不能再换牌。
  - 一旦摸到能胡的牌或有能杠的牌，都必须立即执行，不能放弃。

### 其他重要规则
- **漏胡**：如果“敲”了之后，对家打出的能胡的牌你放弃了，那么在同一轮中，其他人再打出**同一张牌**，你也不能胡。只有等你下次摸牌或碰杠后，这个限制才会解除。
- **一炮多响**：允许一张点炮牌被多个玩家同时胡牌。
- **抢杠**：当其他玩家碰牌后，摸到同一张牌进行“加杠”时，如果你正好胡这张牌，可以“抢杠”。抢杠按**自摸**计算分数，但由进行杠牌的那一家**独自支付三家**的分数。

## 2. 计分方法

上海敲麻的计分由“花”和“番”两个维度构成，最终分数可能有“勒子”封顶。

### 2.1 花的计算
“花”是基础分数，来源于手中的特定牌张或杠牌，每朵花计1分。

| 类型                | 具体牌型                                         | 得花数量                     |
| ------------------- | ------------------------------------------------ | ---------------------------- |
| **花牌**            | 春、夏、秋、冬、梅、兰、竹、菊（每张）           | **1花**                      |
| **风牌刻子/杠**     | 东、南、西、北（刻子/明杠/暗杠）                 | 刻子1花，明杠2花，暗杠3花    |
| **箭牌刻子/杠**     | 中、发、白（刻子/明杠/暗杠）                     | 刻子2花，明杠3花，暗杠4花    |
| **万/筒/条杠**      | 明杠/暗杠                                         | 明杠1花，暗杠2花              |

### 2.2 番种的番数
“番”是对特定胡牌牌型的奖励倍数。**不同平台或朋友局番数可能略有差异**，建议游戏前先确认。以下是常见番种及其番数（或勒子数）：

| 胡牌牌型       | 番数/奖励                                  | 简要说明                                     |
| -------------- | ------------------------------------------ | -------------------------------------------- |
| **门清**       | **1番** 或 **2番**                         | 没有吃、碰、明杠，全靠自己摸牌听牌。         |
| **平胡**       | **1番**                                    | 最基础的胡牌牌型（4组顺子/刻子+1对将牌）。   |
| **碰碰胡**     | **1番** 或 **2番**                         | 由4组刻子或杠子组成。                        |
| **混一色**     | **1番** 或 **2番**                         | 由一种花色序数牌加字牌组成。                 |
| **杠开**       | **1番** 或 **2番**；**不足勒子按勒子算**   | 杠牌后补的那张牌正好胡牌。                   |
| **大吊车**     | **1番** 或 **2番**；**不足勒子按勒子算**   | 吃碰杠后，手中只剩一张牌单吊胡牌。           |
| **抢杠**       | **按自摸计，输分由杠牌者包付**             | 抢别人的加杠而胡牌。                         |
| **清一色**     | **1个勒子**                                | 全部由一种花色的序数牌组成。                 |
| **混碰**       | **1个勒子** 或 **4番**                     | 混一色 + 碰碰胡。                            |
| **风一色**     | **2个勒子**                                | 全部由风牌组成（可乱序）。                   |
| **清碰**       | **2个勒子**                                | 清一色 + 碰碰胡。                            |
| **八花齐**     | **2个勒子**                                | 起手或摸牌过程中凑齐全部8张花牌，可立即胡牌。|
| **风碰**       | **4个勒子**                                | 全部由风牌组成的碰碰胡。                     |

### 2.3 勒子（封顶）
- **定义**：“勒子”是单局输赢的**封顶值**。通常规定 **1个勒子 = 10花**。例如，约定每花1元，则1勒子=10元，一局输赢不超过10元。
- **特殊规则对勒子的影响**：
  - **杠开、大吊车**：如果按花和番计算后不足一个勒子，会**直接提升为一个勒子**计算。
  - **荒番、开宝**（全局翻倍）：
    - **荒番**：上一局流局，下一局进入“荒番”状态，该局所有人的输赢**翻倍**。连续流局可累加翻倍。
    - **开宝**：掷骰子时，若两个骰子**点数相同**或点数为**1和4**，该局即为“开宝”局，输赢同样**翻倍**。
  - 翻倍效果可与牌型本身的勒子数叠加。例如，在“荒番”局胡了一个“清碰”（2勒子），最终按 **4个勒子** 计算。

### 2.4 支付规则
- **点炮胡**：谁点炮，谁**单独**支付胡牌者应得的全部分数。
- **自摸胡**：胡牌者从其他**三家**各收取一份分数。

> **注意**：线上平台或线下朋友局的具体细节（如番数、勒子大小、是否加入“无花果”等变体）可能略有差异，游戏前请先确认统一规则。
